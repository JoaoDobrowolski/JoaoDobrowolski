### <div align="center">👣 About Me:</div>
<div align="justify">
<p>
Web Developer, 25 years old, from Brazil, Mechanical Engineer. Since childhood, always had an aspiration to automate repetitive tasks. The first climb on the programming mountain began at the start of the Mechanical Engineering degree at <a href="http://www.utfpr.edu.br/" target="_blank">UTFPR</a>, following the trails of <a href="https://www.w3schools.com/cpp/cpp_intro.asp" target="_blank">C++</a> and soon after, <a href="https://www.python.org/" target="_blank">Python</a> and <a href="https://www.w3schools.com/java/java_intro.asp" target="_blank">Java</a>. However, the view was still foggy, and what cleared the skies was the course at <a href="https://www.betrybe.com/" target="_blank">Trybe</a>, which paved the road to the top with <a href="https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Introduction" target="_blank">JavaScript</a> and <a href="https://www.typescriptlang.org/" target="_blank">TypeScript</a>, venturing further along the path with <a href="https://www.python.org/" target="_blank">Python</a>, almost always accompanied by the guides <a href="https://www.techtarget.com/searchsoftwarequality/definition/agile-software-development" target="_blank">Agile Methodologies</a>, <a href="https://wearecommunity.io/communities/epam-poland/articles/1190#:~:text=your%20systems%20correctly.-,SOLID%20is%20an%20acronym%20for%20five%20main%20principles%20of%20Object,principle%20and%20dependency%20inversion%20principle." target="_blank">SOLID</a>, and <a href="https://www.techtarget.com/searchapparchitecture/definition/object-oriented-programming-OOP" target="_blank">OOP</a>, while staying hydrated with a lot of learning in <a href="https://www.forbes.com/advisor/business/soft-skills-examples/" target="_blank">Soft Skills</a>. Every step of this climb was accompanied by a project, with more than 30 in total, some of which were group projects. The most notable tools used for climbing these steps were <a href="https://react.dev/" target="_blank">React</a>, <a href="https://react-redux.js.org/" target="_blank">Redux</a>, <a href="https://legacy.reactjs.org/docs/context.html" target="_blank">Context API</a>, <a href="https://nodejs.org/en" target="_blank">NodeJS</a>, <a href="https://expressjs.com/" target="_blank">Express</a>, <a href="https://sequelize.org/" target="_blank">Sequelize</a>, <a href="https://www.docker.com/" target="_blank">Docker</a>, <a href="https://www.mysql.com/" target="_blank">mySQL</a>, <a href="https://www.mongodb.com/" target="_blank">mongoDB</a>, <a href="https://git-scm.com/" target="_blank">GIT</a>, <a href="https://github.com/" target="_blank">GitHub</a>, among others. Testing quality was ensured with <a href="https://jestjs.io/" target="_blank">Jest</a>, <a href="https://testing-library.com/docs/react-testing-library/intro/" target="_blank">RTL</a>, <a href="https://mochajs.org/" target="_blank">Mocha</a>, and <a href="https://www.chaijs.com/" target="_blank">Chai</a>.<br>
In recent years, the climb continued with challenging projects at <a href="https://olby.com/" target="_blank">Olby</a>, where collaboration with the team led to significant achievements. The most recent summit was reached with a project for <a href="https://scholarsoffinance.org/" target="_blank">Scholars of Finance</a>, a non-profit organization, where <a href="https://nextjs.org/" target="_blank">NextJS</a> was the tool of choice to build efficient solutions. The journey up the mountain persists, always exploring new technological trails and striving for greater heights.
</p>
